# Portal Guide - Microsoft Foundry 포털 기반 실습

Microsoft Foundry Portal의 UI를 사용하여 리소스를 생성하고 관리하는 실습 가이드입니다.

## 📖 포털 기반 실습 특징

- ✅ **코드 작성 불필요**: 웹 UI를 통해 모든 작업 수행
- ✅ **시각적 학습**: 각 단계별 스크린샷 제공
- ✅ **빠른 시작**: 개발 환경 설정 없이 바로 시작
- ✅ **개념 이해**: Foundry의 기능과 구조를 직관적으로 학습

## 📚 실습 모듈

1. **[환경 설정](./01-setup.md)** - Resource Group 및 Foundry 리소스 생성
2. **[모델 및 배포](./02-models.md)** - 모델 배포 및 Model Router 구성
3. **[에이전트 개발](./03-agents.md)** - 다양한 에이전트 구축 및 배포
4. **[Foundry IQ](./04-foundry-iq.md)** - AI Search 및 지식 베이스 구축
5. **[워크플로우](./05-workflows.md)** - Sequential, Group Chat, Human-in-loop 워크플로우
6. **[평가](./06-evaluations.md)** - 에이전트 성능 평가
7. **[Control Plane](./07-control-plane.md)** - 운영 모니터링 및 관리

## 🚀 시작하기

1. **[01-setup.md](./01-setup.md)**부터 순차적으로 진행하세요
2. 각 모듈의 스크린샷을 참고하여 Portal에서 작업하세요
3. 이전 모듈에서 생성한 리소스를 다음 모듈에서 활용합니다
